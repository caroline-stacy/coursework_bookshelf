📚 О приложении

Книжная полка — это персональный каталог книг для отслеживания прочитанного. Приложение позволяет вести список книг с оценками, цитатами и обложками, сохраняя всё прямо в браузере без регистрации и серверов.


Все данные хранятся в localStorage вашего браузера и никуда не отправляются.

📖 Инструкция по использованию

  • Добавить книгу — нажмите кнопку «Добавить книгу» на главной или перейдите в раздел «+ Добавить». Заполните обязательные поля: название, автор, жанр и статус. Вставьте URL обложки — появится предпросмотр.
  • Поиск и фильтрация — используйте строку поиска для поиска по названию, автору или жанру. Кнопки-фильтры позволяют показывать только книги с нужным статусом.
  • Отметить прочитанной — поставьте галочку «Прочитано» внизу карточки книги. Статус обновится и сохранится автоматически.
  • Удалить книгу — нажмите кнопку ✕ в правом углу карточки. Появится диалог подтверждения. Удалённые книги не восстанавливаются.
  • Статистика — в разделе «Статистика» можно увидеть распределение по жанрам, рейтингам и список любимых цитат из ваших книг.

⚙️ Технологии

  • HTML5 — семантическая вёрстка
  • CSS3 — Flexbox, Grid, CSS-переменные
  • JavaScript ES6+ — чистый JS, без фреймворков
  • localStorage — хранение данных между сессиями
  • Google Fonts — Playfair Display, Lora, DM Mono
